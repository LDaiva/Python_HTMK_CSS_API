WS API kliento adaptavimas

1. WS API
SQLAlchemy ORM /užklausomis? gauna F025 formos registracijos duomenis iš HIS DB ir atvaizduoja F025 formoje.
2. WS API
išsaugo F025 formoje suvestą apsilankymo paslaugų informaciją ir sukuria XML SOAP žinutę.
3. WS API
siunčia F025 formos duomenų pranešimo HTTPS SOAP užklausą (angl. request) ir autentifikacijos parametrus
APAP HTTP WS serveriui.
4. APAP WS serveris
autentifikuotam klientui suformuoja XML failą ir grąžina atsakymą (angl. response) apie F025 formos išsaugotus duomenis.
5. WS API apdoroja gautą SOAP žinutę ir perduoda duomenis į HIS DB
WS API autentifikavimas
Vartotojo vardas ir slaptažodis turi būti perduodami kiekvienoje užklausoje HTTP(S) header srityje parametro-reikšmės pavidalu (key-value, parametrai „username“ ir „password“).
